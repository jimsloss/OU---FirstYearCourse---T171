The GUI effect
The Graphical User Interface was a key element in the success of the PC and the Web.  Before the GUI the PC and the Internet were both realms for the computer literate, needing specialised knowledge in order to accomplish anything worthwhile from them.  The GUI brought usabilty and familiarity to these areas, opening the door for a new world of users.

The Web effect
Before the Web the Internet was far from user-friendly, usage was limited mainly to computer experts, engineers, scientists, and librarians.  The web allowed the average person to access and use the Internet, without the need for specialist knowledge or terminals.  Its hypertext nature and sheer volume of information makes the Web a highly usable and useful tool, so much can now be achieved from a simple home-computer that they are fast becoming a necessary household appliance.  There is also much to gain for any business having a net-presence.

Similarities & Differences

Using a PC with a graphical interface has many similarities to using the Web;

  • Whichever computer or application you use, or if sending/receiving email, visiting sites on the net, or compiling a web page, all of these have a familiar setting.  Those menus, toolbars, icons, and helpscreens are always there, a benefit from protocols and standards.
  • When you search for a file on your hard disc you specify a path to it, in the same way that the URL locates a web page, however the URL normally provides that path for you, whereas with your pc you have to negotiate the route yourself, via a series of selections with a mouse.
  • You can find a file on your hard drive by using the search facilities provided, search engines provide this facility for the web.  Your PC can find a specific file in seconds, however locating a file on the net can prove much more time consuming.  To narrow your findings you need to be proficient in using the webs search facilities.
It is a common mistake to see the Web as a graphical interface to the Internet, it is not.  The Internet plays host to the Web, allowing it to function via its network of networked computers, however the Web itself is a system of protocols and hypertext links, a means of working, shopping, communicating, advertising, researching, and even entertainment, which can be accessed via any interface to the Internet.

Assessing possible future developments

Computing and networking has come a long way, with major changes having taken place enroute, it would therefore not seem inappropriate to think that major changes are ahead as well.

  • The Molecular Electronics Circuit and similar advancements could led to integrated web access via everyday objects.
  • The Internet is likely to be linked to television, which should have considerable effect on the PC.
  • The Semantic Web would provide a new form of Web content that is meaningful to computers, proving these views correct in part: Future Possibilities and Network possibilities.
  • Other possible methods of input could lead to a GUI overhaul.
